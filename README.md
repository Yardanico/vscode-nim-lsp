# vscode-nim-extension 

vscode nim extension using LSP server write in nim

## Features

* definition  
* executeCommand  <kbd>F6</kbd>
* hover  
* references  
* syntax Highlight  
* completion  

## Requirements

[nimlsp](https://github.com/PMunch/nimlsp) executable can be found in `PATH`


## Known Issues
